# Gestor de agenda de contactos 
Descripción Breve

Este sistema es un gestor de contactos desarrollado en C++ que permite agregar, buscar, ordenar, eliminar y almacenar contactos utilizando diferentes estructuras de datos, como: vector, lista enlazada, pila, cola, árbol binario de búsqueda

Incluye además funciones avanzadas como búsquedas secuenciales y binarias, historial de operaciones, contactos pendientes y recorridos del árbol (inorden y preorden).

Requisitos Técnicos
Lenguaje: C++
